20基于springboot+vue的农产品销售系统

【技术栈】
1.架构：B/S、MVC
2.系统环境：windows/mac
3.开发环境：IDEA2021.2，JDK1.8、Maven、Mysql,tomcat
4.技术栈：java,Mysql,SprintBoot,vue,tomcat

【功能模块】
每个项目都有登陆功能，分普通用户，商家及管理员。具体功能可见结构图
